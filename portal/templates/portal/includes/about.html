À propos du projet OGSL

Le projet OGSL (Observatoire Global du Saint-Laurent) a été développé dans le cadre du cours INF37407 – Technologie de l’inforoute à l’Université du Québec à Rimouski (UQAR). Il vise à regrouper et visualiser des données ouvertes canadiennes issues de plusieurs portails tels que OpenGouv, CanWin, Données Québec et Boréalis.

Le site démontre l’intégration d’API CKAN et Dataverse, ainsi que la mise en œuvre d’une cartographie interactive basée sur Leaflet.js. Ce projet illustre les compétences en développement web avec Django, en visualisation et exploitation de données ouvertes.

Python / Django Leaflet.js CKAN API Dataverse API Bootstrap 5 Playwright
